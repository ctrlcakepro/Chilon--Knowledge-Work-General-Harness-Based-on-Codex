## 社会学习理论复习笔记

### 1. 一句话主线

Bandura 的社会学习理论认为，人并不只是通过直接强化来学习，也会通过观察他人的行为及其后果来学习。

### 2. 核心观点

| 要点 | 说明 |
|---|---|
| 观察学习 | 个体通过观察榜样的行为和结果来形成新的行为。 |
| 榜样作用 | 被观察的人或角色会成为学习对象，例如父母、同伴、教师、媒体人物。 |
| 替代强化 | 看到别人因为某种行为受到奖励或惩罚，会影响自己是否模仿该行为。 |
| 认知因素 | 学习不是机械刺激反应，注意、记忆、预期和自我调节都很重要。 |
| 自我效能感 | 个体相信自己能否完成某事，会影响动机、坚持和表现。 |

### 3. 观察学习的四个过程

1. 注意：学习者先注意到榜样行为。
2. 保持：学习者把行为表征保存在记忆中。
3. 再现：学习者有能力把观察到的行为表现出来。
4. 动机：学习者有理由去执行这个行为，例如期待奖励、避免惩罚或认同榜样。

### 4. 强化在理论中的作用

Bandura 并没有否认强化的重要性，但他认为强化不是学习发生的唯一条件。行为主义更强调直接强化，而社会学习理论强调：即使没有亲自被奖励或惩罚，人也可以通过观察别人被奖励或惩罚来调整自己的行为。

### 5. 和行为主义的区别

| 维度 | 行为主义 | 社会学习理论 |
|---|---|---|
| 学习来源 | 直接经验、刺激-反应、强化 | 直接经验 + 观察他人 |
| 是否强调认知 | 较少强调内部心理过程 | 强调注意、记忆、预期、自我调节 |
| 强化作用 | 学习的核心机制 | 影响行为表现和动机，但不是唯一学习来源 |
| 人的角色 | 更像被环境塑造 | 既受环境影响，也能主动观察、判断和调节 |

### 6. 考前速记版

社会学习理论 = 观察别人也能学。  
核心人物：Bandura。  
关键词：观察学习、榜样、替代强化、自我效能感。  
四过程：注意、保持、再现、动机。  
和行为主义区别：不是只靠直接强化，还强调认知和观察。

### 7. 易混点

不要把社会学习理论理解成完全反对行为主义。它更像是在行为主义基础上加入观察学习和认知因素，说明人可以通过社会环境中的榜样和后果信息来学习。
