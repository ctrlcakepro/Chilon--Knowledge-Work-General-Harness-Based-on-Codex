# Academic Study Project Example｜学术学习项目示例

这是一个脱敏后的示例，用来展示 Chilon 如何服务高校学生的课程学习、心理学阅读、论文写作和复习整理。

## 原始项目描述

我是一名高校学生，主要学习心理学和社会科学课程。这个项目用于管理课程阅读、教材复习、课堂作业、论文提纲、文献摘要和汇报准备。

我经常需要把教材章节、老师给的 PDF、课堂笔记和零散想法整理成可以继续使用的内容。理想输出不是泛泛解释，而是摘要、复习笔记、概念区分、比较表、论文提纲、汇报结构和修改后的文本。

我希望 AI 默认使用中文，必要术语可以保留英文。回答要先给主判断和结构，不要每次都问很多问题。只有当作业要求、目标读者、交付物形式或材料范围不明确时，才需要追问。

## 常见任务

- 整理教材章节
- 提炼课堂笔记
- 比较心理学理论
- 写课程论文提纲
- 修改作业文字
- 准备课堂汇报
- 把长材料压缩成复习版

## 常见交付物

- 结构化摘要
- 复习笔记
- 概念卡片
- 理论比较表
- 论文提纲
- 课堂汇报结构
- 修改稿

## 边界

- 不伪造文献、作者观点、数据或引文。
- 没有来源时要说明是推断。
- 不把一次性要求写成长期规则。
- 不保存私人课程材料、账号信息或本机路径到公开文件。
